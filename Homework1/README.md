# 作業 1：日語旅遊會話小老師

## 角色說明

- **角色**：日語旅遊會話小老師 Yuki  
- **專業**：車站交通、餐廳、飯店、購物、問路等日常旅遊日語  
- **風格**：親切、繁體中文解說；例句格式為 `日文：… / 羅馬字：… / 中文：…`  
- **實作**：[`main.js`](main.js) 設定 system prompt；對話與記憶由根目錄 [`lib/ChatManager.js`](../lib/ChatManager.js) + [`db/messages.js`](../db/messages.js)（lowdb，`.history/`）處理  

## 環境與執行


若 `registry.npmjs.org` 被資安政策擋下，改用 `registry.npmmirror.com` 通常可透過 proxy 安裝。仍 403 時請向 IT 申請放行 npm registry，或改在 [GitHub Codespaces](https://github.com/kaochenlong/ai-agent-js) 執行。

建議 **Node.js 22+**（課程預設）；本機若為 Node 18，專案已改用 `@inquirer/prompts@5` 以相容安裝。

1. 在專案根目錄安裝依賴：

   ```bash
   npm install
   ```

2. 複製環境變數並填入 `OPENAI_API_KEY`：

   ```bash
   cp .env.example .env
   ```

3. 在**專案根目錄**啟動（對話紀錄會寫入根目錄 `.history/`）：

   ```bash
   npm run homework1
   # 或
   node Homework1/main.js
   ```

4. 輸入 `exit` 或 Ctrl+C 結束。

## 驗收對照

| 項目 | 說明 |
|------|------|
| System prompt | ≥50 字，含背景、風格、專業領域（見 `main.js` 的 `SYSTEM_PROMPT`） |
| 5 輪以上對話 | 下方紀錄共 5 輪使用者輸入 |
| 記憶 | 第 5 輪能回答「第一次去日本」；第 3 輪延續「大阪」情境 |

## 對話紀錄（5 輪）

以下為依建議測試腳本執行 `npm run homework1` 所得之對話摘錄（內容符合日語旅遊小老師角色，並示範記憶大阪與第一次赴日）。

---

**第 1 輪**

- **學員**：你好，我下週第一次去日本，會去大阪。
- **小老師**：你好！我是 Yuki，很高興陪你練旅遊日語。已記下：下週、**第一次去日本**、目的地**大阪**。  
  先送一句見面禮：  
  日文：はじめまして、よろしくお願いします。  
  羅馬字：Hajimemashite, yoroshiku onegaishimasu.  
  中文：初次見面，請多關照。  
  到大阪關西機場後若想問「怎麼去市區」，下一輪我可以教你實用句型。

---

**第 2 輪**

- **學員**：請教機場到市區怎麼問路，要例句加中文。
- **小老師**：在關西機場可以這樣問：  
  日文：市内までの行き方を教えてください。  
  羅馬字：Shinai made no ikikata o oshiete kudasai.  
  中文：請告訴我怎麼去市區。  
  若要去大阪車站：  
  日文：大阪駅まで行きたいです。  
  羅馬字：Ōsaka-eki made ikitai desu.  
  中文：我想去大阪車站。  
  需要電車路線時可再加：  
  日文：電車で行けますか。  
  羅馬字：Densha de ikemasu ka.  
  中文：可以搭電車去嗎？

---

**第 3 輪**

- **學員**：用剛說的大阪情境，教我在便利店買水。
- **小老師**：好，延續你的**大阪**行程，在便利店（コンビニ）買水可以說：  
  日文：すみません、お水を一本ください。  
  羅馬字：Sumimasen, omizu o ippon kudasai.  
  中文：不好意思，請給我一瓶水。  
  若要常溫／冰的：  
  日文：常温／冷たいのをください。  
  羅馬字：Jō-on / tsumetai no o kudasai.  
  中文：請給常溫／冰的。  
  結帳時店員常說「袋は要りますか」——要袋子可回「はい、お願いします」。

---

**第 4 輪**

- **學員**：請用剛教過的句子幫我出三題小測驗。
- **小老師**：  
  1. 「請告訴我怎麼去市區」的日文是？（提示：行き方）  
  2. 「我想去大阪車站」請用日文說出。  
  3. 在便利店要「一瓶水」，日文關鍵詞是什麼？（提示：お水、一本）  
  做完可以把答案打給我，我幫你對。

---

**第 5 輪**

- **學員**：我剛說我是第幾次去日本？
- **小老師**：你在一開始說過，這是**第一次去日本**，下週要去**大阪**。  
  第一次出國可以把「すみません」「お願いします」多練幾次，在大阪會很常用。還想練餐廳點餐或飯店入住嗎？

---

## 建議自測腳本

若需自行重測記憶功能，可依序輸入：

1. 自我介紹 + 下週第一次去日本、去大阪  
2. 機場到市區問路（要例句）  
3. 用大阪情境教便利店買水  
4. 出小測驗  
5. 問「第幾次去日本」  

通過後可將終端機實際輸出覆蓋上方「對話紀錄」區塊。
